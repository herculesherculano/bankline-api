<h1 align="center"> Bankline API </h1>

<h2 align="justify">API que simula as movimentações de um correntista em um banco, onde é possível cadastrar correntistas, criar receitas e despesas, e listar todas as movimentações de um correntista.</h2>

<h2 align="justify"> Projeto desenvolvido durante a Santander DevWeek com a DIO, usando as tecnologias Java e SpringBoot. </h2>

## Deploy da API no Heroku:

> https://hercules-dio-bankline-api.herokuapp.com/swagger-ui.html
